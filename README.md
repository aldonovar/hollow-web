# HOLLOW BITS Web

Sitio oficial de **HOLLOW BITS**, el DAW desktop-first impulsado por **ALLYX** y desarrollado junto a **Ethereal Sounds**.

La web esta construida como escaparate tecnico + comercial:

- narrativa de producto completa
- validacion de confiabilidad explicada en lenguaje claro
- identidad visual premium con inspiracion surreal sutil
- experiencia fluida con Lenis + GSAP + parallax + SVG animation

## Stack

- React + TypeScript + Vite
- GSAP + ScrollTrigger
- Lenis smooth scroll

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Estructura base

- `src/App.tsx`: estructura principal del sitio y animaciones
- `src/content.ts`: contenido tecnico/comercial del producto
- `src/index.css`: sistema visual, layout y motion styles

## Deploy recomendado

1. Subir repo a GitHub (`aldonovar/hollow-web`)
2. Importar el repo en Vercel
3. Framework preset: Vite
4. Build command: `npm run build`
5. Output directory: `dist`
